## üé® Formatting Conventions

### Headings:
- H1 (#) - Page title only
- H2 (##) - Major sections
- H3 (###) - Subsections
- H4 (####) - Sub-subsections

### Code Blocks:
```bash
# Terminal commands
```

```typescript
// TypeScript/JavaScript
```

```css
/* CSS */
```

### Callouts:
> **Note:** Important information

> **Warning:** Critical information

> **Tip:** Helpful hint

### Lists:
- Use hyphen (-) for bullet points
- Use numbers for ordered lists
- Use proper indentation
